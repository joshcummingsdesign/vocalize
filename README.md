# Vocalize

Voice control for your computer with a Vim mindset.

## Requirements

- python ^3.10.0
- pipenv ^2021.5.0

## Getting Started

1.  Download the Kaldi model

        ./scripts/download-model.sh

2.  Install the project dependencies

        pipenv install

3.  Actiate the virtual environment

        pipenv shell

4.  Run the demo

        python demo.py
